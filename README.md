# Portfolio

Personal portfolio website.

## Tech Stack

- React 19
- TypeScript
- Vite
- Oxlint
- Lucide React

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm

### Setup Commands

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build production bundle
npm run build

# Run linter
npm run lint
```
